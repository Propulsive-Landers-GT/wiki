# Fitting Installation Practices

This page is meant to collect basic installation practices for the fittings we commonly use on GTPL systems. Always check the manufacturer datasheet for the exact fitting you are using. Though, the general idea is pretty much the same: identify the fitting type first, understand how it seals, then install it using the correct method.

### NPT Fittings

NPT fittings seal through tapered threads. As the fitting tightens, the threads wedge together and create the seal.

For NPT fittings (ones that you need to teflon):

* Always use PTFE tape or an approved thread sealant unless the fitting is specifically designed to seal dry.
  * Wrap PTFE tape in the same direction that the fitting tightens so the tape does not unwrap during installation.
  * Use about 2-4 wraps of PTFE tape for most small fittings.
  * Start the tape one or two threads back from the end of the fitting so tape does not shred into the flow path.
  * Keep the tape smooth and tight around the threads.
  * Do not let loose tape pieces enter the system.
  * For oxygen service, only use sealants or tape that are explicitly rated for oxygen compatibility.
* Do not over-tighten NPT fittings. The tapered threads can crack ports, manifolds, regulators, or valve bodies if forced too far.
* Avoid repeatedly removing and reinstalling the same NPT joint if possible.
  * Speaking from experience we've almost always had to re-teflon after we loosen an NPT fitting so keep that in mind.&#x20;

### Flared and AN Fittings

Flared fittings seal on the angled flare surface and not on the threads. Threads on there just provide the clamping force but it's the flare that does the sealing.&#x20;

For flared or AN fittings:

* Inspect the flare surface before installation.
* Do not use thread tape on the sealing face.
* Make sure the flare angle and fitting type match.
* Tighten enough to seat the flare, but do not crush or deform it.
* If it leaks, inspect the flare instead of just tightening harder.
