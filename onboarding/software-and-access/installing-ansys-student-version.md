# Installing Ansys Student Version

Ansys Student is the free student version of Ansys. For GTPL, this is usually enough for learning Workbench, Mechanical, Fluent, meshing, and basic CFD/FEA with a 512K mesh limit. If you are working on larger team simulations, you may eventually need the full Ansys version instead.

Official download link: [https://www.ansys.com/academic/students/ansys-student](https://www.ansys.com/academic/students/ansys-student)

As of the latest Ansys page, the current student download is **Ansys Student 2026 R1**.

### Before Installing

Before you start:

* Make sure you are on Windows 10 or Windows 11, 64-bit.
* Make sure you have administrator privileges on your computer.
* Make sure you have enough storage space. Ansys lists 50 GB as the minimum hard drive space.
* If you already have another Ansys version installed, be careful. The Student installer may overwrite existing Ansys installations.

### Installation Steps

1. Go to the official Ansys Student download page:\
   https://www.ansys.com/academic/students/ansys-student
2. Click **Download Ansys Student**.
3. Once the download finishes, extract/unzip the downloaded folder.
4. Open the extracted folder.
5. Right-click `setup.exe`.
6. Select **Run as administrator**.
7. Read and accept the license/clickwrap agreement.
8. Continue through the installer using the default options unless you know you need to change something.
9. When the installer finishes, close it.
10. Restart your computer.
11. After restarting, open Ansys from the Start Menu by searching for:

```
Workbench
```
